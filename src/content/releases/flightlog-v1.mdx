Welcome to the very first entry of **FlightLog**, our breathtaking, open-source Changelog engine.

### The Problem

SaaS companies need a place to announce product updates, but setting up a dedicated blog is overkill, and dropping notes in a Discord channel gets lost instantly. 

### The Solution

FlightLog provides a beautiful, linear-style timeline built specifically to host high-conversion release notes. 

- **Markdown Native:** Write your releases in standard `.mdx`.
- **Custom Badges:** Highlight Major Releases, Updates, or Bug Fixes.
- **Brand Aligned:** Seamlessly integrates with the rest of the Scramjet OSS aesthetic.

You are reading this release note *inside* FlightLog right now! Want to deploy your own? Grab the code on [GitHub](https://github.com/scramjetio/flight-log).
