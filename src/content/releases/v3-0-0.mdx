Welcome to Nova OS v3.0. This major release represents six months of engineering effort entirely focused on the underlying architecture. While the UI remains familiar, the engine beneath it is completely new.

### Architectural Changes

- ☁️ **Cloud Architecture Upgrade:** We migrated the entire core to a globally distributed edge network. This increases speed and reliability, dropping average global latency to under 30ms.
- 🔌 **New Integration Ecosystem:** We've launched the Integration Hub with 15+ new services added on day one, including seamless webhooks and data firehoses.
- 🛡️ **Security Hardening:** Implemented Enterprise SSO (SAML/OIDC) and upgraded our at-rest encryption protocols to AES-256-GCM.
