We've been listening closely to your feedback over the last few weeks. The v2.9 release is entirely dedicated to making your day-to-day interactions smoother and more intuitive.

### Highlights

- ✨ **Sleek New Dashboard Interface:** We reduced the visual noise across all main panels. Data tables now have better contrast and typography for increased readability.
- ⚡ **Faster Data Syncing:** Streamlined the telemetry ingest pipeline. Data syncing is now noticeably faster, especially on high-latency connections.
- 📄 **Updated Reporting Tools:** The PDF export engine has been rewritten. Reports now generate in half the time and support custom branding.
